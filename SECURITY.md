# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 0.x     | ✅        |

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub Issues.**

If you discover a security vulnerability in `inkshot-engine`, please report it responsibly by opening a [GitHub Security Advisory](https://github.com/PukoDeveloper/inkshot-engine/security/advisories/new) (click "Report a vulnerability").

Please include as much of the following information as possible to help us triage and resolve the issue quickly:

- Type of vulnerability (e.g. prototype pollution, ReDoS, unsafe deserialization)
- The affected file(s) and line numbers
- A minimal reproduction case or proof-of-concept
- Potential impact and severity

We will acknowledge your report within **72 hours** and aim to release a patch within **14 days** for confirmed vulnerabilities.
